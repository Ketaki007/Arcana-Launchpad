# Purpose

Describe what users are trying to accomplish.

---

# Primary Goals

- Understand portfolio performance
- Compare investment opportunities
- Investigate anomalies
- Understand risk and exposures
- Evaluate scenarios
- Validate investment hypotheses
- Monitor portfolio health
- Discover actionable insights

---

# Design Implications

Interfaces should minimize cognitive effort while
supporting deep analytical thinking.

Users should never lose context during exploration.

---

# AI Interpretation

Before selecting patterns or components,
identify which user goal is being supported.

If multiple goals exist,
prioritize the primary decision-making task.
