# Purpose

Maintain consistent language throughout the product.

---

# Terminology Principles

- Use financial terminology consistently.
- Prefer product-specific vocabulary over generic alternatives.
- Avoid ambiguous labels.
- Match language used by investment professionals.

---

# Preferred Terms

| Preferred | Avoid |
|------------|-------|
| Portfolio | Account |
| Benchmark | Reference |
| Performance Attribution | Performance Breakdown |
| Factor Exposure | Exposure Details |
| Scenario Analysis | Simulation |

---

# Design Implications

Terminology influences user confidence.

Changing labels for similar concepts increases
cognitive load.

---

# AI Interpretation

Always prefer established product terminology
when generating interfaces or documentation.

Do not invent new labels unless explicitly instructed.
