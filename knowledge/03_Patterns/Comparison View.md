# Purpose

Enable users to identify meaningful differences between related entities.

---

# Suitable For

- Portfolio vs Benchmark
- Portfolio vs Portfolio
- Scenario Comparison
- Factor Comparison

---

# Design Guidelines

- Align comparable information.
- Highlight differences.
- Preserve shared context.
- Reduce unnecessary navigation.

---

# AI Interpretation

Comparison views should minimize cognitive effort by presenting related information together.
