# Purpose

Buttons enable users to perform actions.

The emphasis of a button should reflect the importance of the action within the current workflow.

---

# Use When

- Completing a workflow
- Confirming an action
- Saving changes
- Running an analysis
- Navigating to the next step

---

# Avoid When

- Displaying information
- Navigating within dense datasets
- Presenting passive content

---

# AI Interpretation

Every screen should have a clear primary action.

Avoid multiple competing primary buttons.
