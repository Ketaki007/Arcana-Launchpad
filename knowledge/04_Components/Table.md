# Purpose

Display structured information that users need to compare, scan, sort or filter.

---

# Use When

- Comparing records
- Reviewing metrics
- Displaying holdings
- Showing attribution reports
- Supporting filtering and sorting

---

# Avoid When

- Showing a small number of summary items
- Storytelling
- High-level dashboards

---

# AI Interpretation

Choose tables when comparison and precision are more important than visual presentation.
