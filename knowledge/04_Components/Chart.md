# Purpose

Visualize relationships, trends and distributions within data.

---

# Use When

- Showing trends
- Comparing changes over time
- Highlighting anomalies
- Visualizing distributions

---

# Avoid When

- Users require exact values
- Data needs sorting or filtering
- Users inspect many records simultaneously

---

# AI Interpretation

Charts should summarize information.

When users need detailed inspection, pair charts with supporting tables.
