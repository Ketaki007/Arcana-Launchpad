# User Flows

## Why This Document Exists

These workflows represent the primary analytical journeys supported by Arcana Launchpad. Features should strengthen these workflows rather than introduce new ones.

---

# Create a Workspace

Goal:
Create a workspace for a new investigation.

Flow:

Workspace
→ Select View
→ Create or Open Page
→ Add Widgets
→ Configure Widgets
→ Link Context
→ Save Page

Supported Features

- Workspace Builder
- Widget Configuration

---

# Investigate a Company

Goal:
Understand a company's performance and investment potential.

Flow:

Open Workspace
→ Search Security
→ Review Company Overview
→ Analyze Financials
→ Compare Peers
→ Review News
→ Capture Insights

Supported Features

- Decision Composer

---

# Resume an Investigation

Goal:
Return to an interrupted investigation with minimal cognitive effort.

Flow:

Open Workspace
→ Workspace Catch-up
→ Review Changes
→ Continue Analysis

Supported Features

- Workspace Catch-up
- Workspace Health

---

# Monitor a Portfolio

Goal:
Maintain awareness of portfolio performance.

Flow:

Open Portfolio View
→ Review Holdings
→ Identify Changes
→ Investigate Outliers
→ Review News
→ Take Action

---

# Compare Securities

Goal:
Evaluate multiple companies within a shared analytical context.

Flow:

Select Security
→ Linked Widgets Update
→ Compare Financials
→ Review Valuation
→ Document Findings
