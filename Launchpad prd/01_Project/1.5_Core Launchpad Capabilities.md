# Core Launchpad Capabilities

## Overview

Arcana Launchpad provides a flexible analytical workspace where users can assemble, organize and monitor financial information through configurable widgets. The platform is designed to support complex, multi-window workflows while allowing analysts to tailor their workspace to individual preferences.

While later sections of this PRD introduce AI-powered capabilities, the following features form the core Launchpad experience.

---

## Workspace Management

- Create, rename and delete workspaces
- Organize workspaces into folders
- Save and restore workspace state
- Duplicate workspaces
- Import / export workspaces

---

## Views & Pages

- Multiple views within a workspace
- Multiple pages within a view
- Drag-and-drop page organization
- Duplicate pages
- Rename pages

---

## Widget Management

- Add widgets
- Remove widgets
- Resize widgets
- Reposition widgets
- Dock and undock widgets
- Maximize / restore widgets

---

## Widget Configuration

- Configure widget settings
- Save widget preferences
- Reset configuration
- Personal defaults

---

## Shared Context

- Link widgets together
- Shared security selection
- Shared date ranges
- Shared comparisons
- Cross-widget synchronization

---

## Layout Management

- Grid layout
- Responsive resizing
- Save layouts
- Restore layouts
- Personal templates

---

## Navigation

- Search within Launchpad
- Recent workspaces
- Recently viewed securities
- Keyboard shortcuts

---

## Collaboration (Optional)

- Share workspace
- Export layout
- Read-only workspace
- Presentation mode

---

## Personalization

- Theme
- Density
- Widget preferences
- Default layouts
